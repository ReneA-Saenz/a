Investigación y Ejemplos de Etiquetas HTML

Etiquetas Básicas

  • <html> ... </html>
    Define el documento HTML y es la raíz de la estructura.
    <html> ... </html>
  • <head> ... </head>
    Contiene metadatos como título, estilos y scripts.
    <head> ... </head>
  • <title> ... </title>
    Define el título que aparece en la barra del navegador.
    <title>Mi Página</title>
  • <body> ... </body>
    Contiene el contenido principal visible de la página web.
    <body> ... </body>
  • <p> ... </p>
    Define un párrafo de texto.
    <p>Este es un párrafo.</p>

    Este es un párrafo.

  • <h1> ... </h6>
    Encabezados jerárquicos, de más a menos importante.
    <h1>Encabezado 1</h1>

    Ejemplo:

    Encabezado 1

    Encabezado 2

    Encabezado 3

  • <br>
    Salto de línea (no tiene cierre).
    Primera línea<br>Segunda línea

    Primera línea
    Segunda línea

  • <a href="URL"> ... </a>
    Crea un enlace a otra página o recurso.
    <a href="https://www.facebook.com/movjuveoficial/?locale=es_LA">Visitar Ejemplo</a>

    Visitar Ejemplo

  • <img src="URL" alt="texto">
    Inserta una imagen.

Etiquetas de Tablas en HTML

Las tablas en HTML se utilizan para mostrar datos tabulares organizados en filas y columnas. Las etiquetas principales para construir una tabla son:

  • <table> ... </table>: Define el contenedor de la tabla.
  • <tr> ... </tr>: Define una fila dentro de la tabla.
  • <th> ... </th>: Define una celda de encabezado (normalmente en la primera fila o columna).
  • <td> ... </td>: Define una celda de datos.
  • <thead> ... </thead>: Agrupa el encabezado de la tabla.
  • <tbody> ... </tbody>: Agrupa el cuerpo principal de la tabla.
  • <tfoot> ... </tfoot>: Agrupa el pie de la tabla, normalmente para totales o notas.

Ejemplo completo de una tabla con todas las etiquetas:

Producto Cantidad Precio Unitario Total
Manzanas 10 Q0.50 Q5.00
Naranjas 5 Q0.80 Q4.00
Plátanos 7 Q0.30 Q2.10
Total General Q11.10

Explicación del ejemplo:

  • <thead> contiene la fila de encabezados con etiquetas <th>.
  • <tbody> contiene las filas con datos usando <td>.
  • <tfoot> contiene una fila con el total general, usando el atributo colspan="3" para unir tres celdas.

Otras etiquetas relacionadas:

  • colspan: Atributo para que una celda abarque varias columnas.
  • rowspan: Atributo para que una celda abarque varias filas.

Etiquetas de Listas

  1. Primer elemento
  2. Segundo elemento

Lista no ordenada (<ul>):

  • Elemento A
  • Elemento B

Lista descriptiva (<dl>):

Etiquetas de Formularios





  • <form> ... </form>: Define un formulario.
  • <input>: Campo interactivo (texto, botón, etc.).
  • <label> ... </label>: Etiqueta descriptiva para controles.
  • <select> ... </select>: Menú desplegable.
